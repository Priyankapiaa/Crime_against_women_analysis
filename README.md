# Crimes Against Women in India — A 21-Year Analysis (2001–2021)

## Project Overview
Analysis of official NCRB government data on crimes against women 
across all Indian states from 2001 to 2021.

## Business Problem
Which states are most affected? Which crimes are growing fastest? 
What policy interventions are needed?

## Tools Used
- Microsoft Excel — Data cleaning & pivot analysis
- SQL Server (SSMS) — Data querying & aggregation
- Power BI — Interactive dashboard

## Key Findings
- Domestic Violence accounts for 39% of all crimes
- UP, Andhra Pradesh, WB and MP are most affected states
- Kidnapping increased 5x over 20 years
- Women Trafficking increased 9x over 20 years
- Dowry Deaths remained unchanged — awareness campaigns working
